# 🎲 Monopoly CLI Game

A fun terminal-based Monopoly-style board game built in Python with colorful UI, dice animations, and multiple gameplay features.

---

## 🚀 Features

- 🎨 Colored ASCII board
- 🎲 Dice animation
- 🏦 Property buying & rent system
- 🎁 Chance events
- 🔒 Jail system (skip turns)
- 💰 High score tracking
- 🏆 Winner animation
- 🖥️ EXE version (no Python required)
